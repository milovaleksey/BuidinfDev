# Быстрый Старт - Система Управления Зданием

## Предварительные требования

1. **PostgreSQL** - база данных
2. **Java 17+** - для бэкенда
3. **Node.js 18+** - для фронтенда
4. **Node-RED** (опционально) - для управления устройствами

## Установка

### 1. Настройка PostgreSQL

```bash
# Создание базы данных
psql -U postgres
CREATE DATABASE building_management;
CREATE USER postgres WITH PASSWORD 'postgres';
GRANT ALL PRIVILEGES ON DATABASE building_management TO postgres;
\q
```

### 2. Запуск Бэкенда

```bash
# Переход в директорию бэкенда
cd backend

# Запуск (автоматическая сборка и запуск)
./start.sh

# Бэкенд будет доступен на http://localhost:8080
```

**Проверка:**
```bash
curl http://localhost:8080/api/auth/health
# Должен вернуть: {"status":"UP"}
```

### 3. Запуск Фронтенда

```bash
# В корневой директории проекта
./start-frontend.sh

# Фронтенд будет доступен на http://localhost:5173
```

### 4. Запуск Node-RED (опционально)

```bash
# Установка Node-RED глобально
npm install -g node-red

# Запуск
node-red

# Node-RED будет доступен на http://localhost:1880
```

## Создание Первого Здания

### Вариант 1: Через UI

1. Войдите как `admin`
2. Перейдите в "Локации"
3. Нажмите "Создать"
4. Выберите "Здание"
5. Введите название и адрес
6. Нажмите "Создать"

### Вариант 2: Через API

```bash
# Получить токен
TOKEN=$(curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin"}' \
  | jq -r '.token')

# Создать здание
curl -X POST http://localhost:8080/api/buildings \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Офис",
    "address": "Москва, ул. Ленина 1"
  }'
```

## Создание Этажа

1. Откройте здание (кликните на него)
2. Нажмите "Создать" → "Этаж"
3. Выберите здание
4. Введите название и номер этажа
5. Нажмите "Создать"
6. Откройте "Редактор" для добавления помещений

## Добавление Помещений

1. Откройте редактор этажа
2. Нажмите "Добавить помещение"
3. Введите данные:
   - Номер помещения
   - Название
   - Тип (офис, коридор, и т.д.)
4. Разместите помещение на плане
5. Измените размеры через синие точки

## Добавление Устройств

1. Выберите помещение
2. Нажмите "Добавить устройство"
3. Выберите тип устройства:
   - Освещение (светильник)
   - СКУД (дверь, турникет)
   - Видеонаблюдение (камера)
   - Кондиционирование (кондиционер)
   - Отопление (батарея)
   - Датчики (температура, CO2)
4. Введите название
5. Устройство появится в помещении

## Отладка

### Проблема: Не могу войти

```bash
# Проверьте бэкенд
curl http://localhost:8080/api/auth/health

# Проверьте БД
psql -U postgres -d building_management -c "SELECT * FROM users;"

# Если нет пользователей, они создадутся автоматически при первом запуске
```

### Проблема: Пустой список зданий

1. Создайте первое здание через UI
2. Или используйте импорт из `/examples/building_sample.json` (если есть)
3. Или подождите - дефолтное здание создастся автоматически

### Проблема: Устройства не управляются

1. Проверьте запущен ли Node-RED: http://localhost:1880
2. Проверьте настройку `nodered.base.url` в `backend/src/main/resources/application.properties`
3. Проверьте логи бэкенда: `tail -f backend/logs/application.log`

### Проблема: Медленная работа

1. Очистите кэш браузера
2. Удалите localStorage: DevTools → Application → Storage → Local Storage → Clear
3. Перезапустите фронтенд

## Структура Ролей

### Admin (Администратор)
- Полный доступ ко всем функциям
- Управление пользователями
- Создание/удаление зданий

### Manager (Менеджер)
- Управление зданиями и этажами
- Редактирование планировок
- Управление устройствами
- Просмотр отчетов

### Operator (Оператор)
- Управление устройствами
- Просмотр планов
- Базовые отчеты

### Viewer (Наблюдатель)
- Только просмотр
- Без прав редактирования
- Без управления устройствами

## Поддержка

При возникновении проблем:

1. Проверьте логи:
   - Бэкенд: `tail -f backend/logs/application.log`
   - Фронтенд: Browser DevTools Console
   - PostgreSQL: `psql -U postgres -d building_management`

2. Перезапустите сервисы:
   ```bash
   # Бэкенд
   cd backend && ./start.sh
   
   # Фронтенд
   ./start-frontend.sh
   ```

3. Сбросьте БД (ВНИМАНИЕ: удалит все данные):
   ```bash
   psql -U postgres -c "DROP DATABASE building_management;"
   psql -U postgres -c "CREATE DATABASE building_management;"
   ```

Готово! Система запущена и готова к использованию. 🎉
